# 🎨 The Ritesh Showcase

A modern, interactive personal portfolio website featuring cutting-edge web technologies, stunning animations, and a showcase of multifaceted talents in development, design, gaming, and content creation.

**🌐 Live Demo:** [The Ritesh Showcase](https://riteshkumar477823-wq.github.io/The-Ritesh-Showcase/)  
**👤 Author:** [Ritesh Kumar](https://www.github.com/riteshkumar477823-wq)

---

## ✨ Features

### 🎯 Core Sections
- **Home** — Eye-catching hero section with animated starfield background and dynamic typing effect
- **About** — Personal introduction with profile card, stats, and social links
- **Passion** — Showcase of three passions: Gaming, Editing, and Web Development
- **Skills** — Technical skill bars and tech stack display
- **Teams** — Carousel of tools and collaborators (GitHub, Gemini, ImageKit, etc.)
- **Projects** — Filterable project grid with live demos and GitHub links
- **Certificates** — Modal gallery of credentials and achievements
- **Contact** — Contact form and direct messaging options

### 🚀 Technology & Animations
- **3D Starfield Canvas** — Interactive parallax star background responding to mouse movement
- **Typed.js** — Dynamic typing animation for role descriptions
- **Owl Carousel** — Smooth carousel for team/tools section
- **Waypoints.js** — Scroll-triggered animations
- **3D Tilt Effects** — Card hover effects with perspective transforms
- **Cursor Glow** — Custom animated cursor effect
- **Intersection Observer** — Smooth element reveal animations on scroll

### 📱 Responsive Design
- Mobile-first approach with breakpoints for all devices
- Hamburger menu for mobile navigation
- Touch-friendly interface
- Optimized performance across screen sizes

## 🔧 How to Use

### 1. **View Online**
Simply visit the [live portfolio](https://riteshkumar477823-wq.github.io/The-Ritesh-Showcase/) in your browser.

### 2. **Local Development**
```bash
git clone https://github.com/riteshkumar477823-wq/The-Ritesh-Showcase.git
cd The-Ritesh-Showcase
# Open index.html in your browser
```

### 3. **Customize**
Edit the following files to personalize:
- `index.html` — Content, text, links
- `style.css` — Colors, fonts, styling
- `script.js` — Animations and interactions
